## Props

| Prop        | Type        | Default    | Description |
|------------|------------|------------|-------------|
| `children` | ReactNode  | —          | Content displayed inside the button (text, icons, or elements). |
| `variant`  | string     | `primary`  | Visual style of the button. <br />Values: `primary`, `secondary`, `outline`, `ghost` |
| `size`     | string     | `md`       | Controls button size. <br />Values: `sm`, `md`, `lg` |
| `disabled` | boolean    | `false`    | Disables the button and prevents interaction. |
| `loading`  | boolean    | `false`    | Shows loading state and disables the button. |
| `className`| string     | `""`       | Adds custom CSS classes to the button. |
| `to`       | string     | —          | Converts the button into a navigation link. |
| `bg`       | string     | —          | Custom background color (hex, rgb, gradient). |
| `color`    | string     | —          | Custom text color. |
| `style`    | object     | —          | Inline styles for custom design. |
| `...props` | object     | —          | Any valid native HTML button attributes. |

### Step 3: Add the Three Core Files
      Inside that new folder, create these three files and paste the code provided in the component's documentation:
      1.  `Button.jsx` (The Logic)

    ### Step 4: Paste in Button.jsx

    ```jsx
    "use client"; // Required for Next.js App Router

import React from "react";
import Link from "next/link"; // Use Next.js Link
import styles from "./button.module.css";

export const Button = ({
  children,
  variant = "primary",
  size = "md",
  disabled = false,
  loading = false,
  className = "",
  to,
  bg,
  color,
  style,
  ...props
}) => {
  const isLink = Boolean(to);
  
  // If it's a link, use Next.js Link component, otherwise use 'button'
  const Tag = isLink ? Link : "button";

  const customStyles = {
    ...style,
    backgroundColor: bg,
    color: color,
  };

  const combinedClasses = `${styles.button} ${styles[variant]} ${styles[size]} ${className}`;

  return (
    <Tag
      // Next.js Link uses 'href', not 'to'
      href={isLink ? to : undefined} 
      className={combinedClasses}
      // Only 'button' tag supports 'disabled' attribute
      disabled={!isLink && (disabled || loading) ? true : undefined}
      style={customStyles}
      {...props}
    >
      {loading ? "Loading..." : children}
    </Tag>
  );
};
    ```

    ### Step 5: Paste in button.module.css

    ```jsx
    /* Base styles - applicable to both <button> and <a> tags */
.button {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  border-radius: 8px;
  font-weight: 500;
  cursor: pointer;
  transition: all 0.2s ease;
  border: none;
  text-decoration: none; /* Ensures Link looks like a button */
  line-height: 1;
  white-space: nowrap;
  outline: none;
}

/* Variants */
.primary {
  background-color: black;
  color: white;
}

.primary:hover:not(:disabled) {
  background-color: #333;
}

.outline {
  background-color: transparent;
  border: 1px solid black;
  color: black;
}

.outline:hover:not(:disabled) {
  background-color: rgba(0, 0, 0, 0.05);
}

.ghost {
  background-color: transparent;
  color: black;
}

.ghost:hover:not(:disabled) {
  background-color: rgba(0, 0, 0, 0.1);
}

/* Sizes */
.sm {
  padding: 0.4rem 0.8rem;
  font-size: 0.8rem;
  height: 32px;
}

.md {
  padding: 0.6rem 1.2rem;
  font-size: 0.9rem;
  height: 40px;
}

.lg {
  padding: 0.8rem 1.6rem;
  font-size: 1rem;
  height: 48px;
}

/* States */
.button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
  pointer-events: none; /* Prevents hover effects on links when disabled */
}

.button:focus-visible {
  box-shadow: 0 0 0 2px white, 0 0 0 4px black;
}
    ```
